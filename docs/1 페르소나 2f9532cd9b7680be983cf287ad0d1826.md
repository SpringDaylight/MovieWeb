## Define - 고객의 문제는? - Today Statement 만들기

"넷플릭스 추천이 평점 기반이라 내 정서적 취향과 안 맞는 영화가 자주 추천된다"

→ 평점이 높아도 내가 좋아하는 분위기나 서사 스타일이 아니면 만족도가 떨어진다

→ 장르만으로는 정서적 경험을 구분하기 어렵다

### 문제 2: 영화 선택 시간이 오래 걸림

"볼 영화를 고르는데 30분 이상 걸린다. 줄거리를 읽어도 내 취향에 맞을지 확신이 안 선다"

→ 시놉시스만으로는 영화의 정서적 톤을 파악하기 어렵다

→ "이 영화가 나한테 맞을까?"라는 질문에 답을 주는 서비스가 없다

### 문제 3: 리뷰 분석의 한계

"리뷰를 10개씩 읽어도 결국 내 취향과 맞는지는 봐야 안다"

→ 다른 사람의 리뷰가 나의 취향과 얼마나 일치하는지 알 수 없다

→ 리뷰어의 취향 성향을 파악하기 어렵다

### 문제 4: 그룹 시청 시 선택의 어려움

"친구들과 함께 볼 영화를 고를 때 각자 취향이 달라서 선택이 어렵다"

→ 여러 사람의 취향을 모두 고려한 추천이 없다

→ 누구는 좋아하고 누구는 싫어할지 미리 알고 싶다

### 문제 5: 감성 검색의 부재

"'잔잔한데 생각 많아지는 영화' 같은 느낌으로 검색하고 싶은데 장르 검색만 된다"

→ 정서적 키워드로 영화를 찾을 수 없다

→ 지금 내 기분에 맞는 영화를 찾기 어렵다

### 문제 6: 추천 이유를 모름

"왜 이 영화가 추천되었는지 이유를 모르겠다"

→ 추천 알고리즘의 근거가 불투명하다

→ 추천을 신뢰하기 어렵다

## Invent - 고객에게 가장 필요한 솔루션은?

### 핵심 솔루션 3가지:

1. **정서 기반 취향 분석 및 만족도 예측**
    - 사용자의 리뷰와 감상 소감을 분석하여 정서적 취향 프로필 생성
    - 특정 영화에 대한 개인별 만족 확률 예측
    - 예측된 확률에 대한 이유를 자연어로 설명
2. **자연어 기반 감성 검색**
    - "우울하지만 따뜻한 결말", "긴장감 있지만 무겁지 않은" 같은 감성 키워드로 검색
    - 사용자의 현재 기분이나 원하는 경험을 자연어로 표현하여 영화 찾기
3. **그룹 취향 시뮬레이션**
    - 여러 사람의 취향을 종합하여 그룹 만족도 예측
    - 각 개인별로 얼마나 만족할지 미리 확인 가능
4. 위의 3가지가 가능하기 위해서는, 기본적인 영화 리뷰 crud가 가능해야 한다.

## Refine - 솔루션에 대한 고객 경험을 어떻게 표현?

### **<고객 니즈>**

- 내 취향을 정확하게 이해하는 추천
- 영화를 보기 전에 만족도를 예측하고 싶음
- 감성적인 키워드로 영화 검색
- 추천 이유를 명확하게 알고 싶음
- 그룹 시청 시 모두가 만족할 영화 찾기

### **<솔루션 아이디어 상세 설명>**

**① 정서 기반 취향 모델링 (A-1)**

사용자가 작성한 리뷰나 한 줄 소감을 LLM으로 분석:

**Input:**

- "영화가 잔잔했는데 계속 생각이 나요. 결말이 열려있어서 여운이 길었어요."

**Process:**

```python
LLM이 텍스트를 분석하여 정서적 선호 추출:
- emotion_tone: ["calm", "melancholic", "warm"]
- narrative_focus: ["character-driven", "relationship"]
- pacing: "slow"
- ending_preference: "open-ended"
```

**Output:**

- 구조화된 사용자 취향 벡터 (JSON 형식)

**ML 작업:**

- 모델 학습 불필요
- LLM의 의미 해석 능력 활용 + 구조화된 출력 스키마

---

**② 영화 특성 벡터링 (A-2)**

영화의 줄거리와 리뷰를 분석하여 정서적 특성 추출:

**Input:**

- 영화 줄거리 (TMDB API)
- 사용자 리뷰 요약 (IMDB/Kaggle)
- 메타 정보 (장르, 평점)

**Process:**

```python
LLM이 영화 텍스트 분석:
줄거리 → "이 영화는 조용한 분위기 속에서 
         인간관계의 변화를 다루며, 
         결말이 명확하지 않고 여운을 남김"

→ 임베딩 생성 (Sentence Transformer)
→ 벡터 DB 저장
```

**Output:**

- 영화별 정서·서사 벡터
- 500-1,000편 수준의 벡터 DB

**ML 작업:**

- Sentence Transformer 임베딩
- 벡터 DB 구축 (Chroma/FAISS)

---

**③ 취향 시뮬레이터 - 만족 확률 계산 (A-3)**

사용자 취향과 영화 특성을 비교하여 만족도 예측:

**Input:**

- 사용자 취향 벡터 (A-1 결과)
- 영화 특성 벡터 (A-2 결과)
- (선택) 유사 사용자 평점 통계

**Process:**

```python
# 차원별 유사도 계산
emotion_sim = cosine_similarity(user.emotion_tone, movie.emotion_tone)
narrative_sim = cosine_similarity(user.narrative_focus, movie.narrative_focus)
ending_match = (user.ending_preference == movie.ending_type) ? 1 : 0

# 가중 합산
score = 0.4 * emotion_sim + 
        0.3 * narrative_sim + 
        0.2 * ending_match + 
        0.1 * user_stat

# 확률로 정규화
satisfaction_probability = sigmoid(score)
```

**Output:**

```python
{
  "satisfaction_probability": 0.78,
  "confidence": 0.71
}
```

**ML 작업:**

- Logistic Regression 또는 Weighted Similarity
- 딥러닝 필수 아님

**④ 설명 가능한 추천 (A-4)**

예측 결과를 자연어로 설명:

Input:

- 만족 확률 (0.78)
- 주요 기여 요소 (emotion_sim: 0.85, narrative_sim: 0.72)

**Process:**

```python
LLM에게 설명 생성 요청:
"당신은 잔잔하고 관계 중심의 서사를 선호하며,
여운 있는 열린 결말을 좋아하는 경향이 있습니다.
이 영화는 해당 특성이 강해 78%의 만족 확률이 예측됩니다."
```

Output:

- 사용자 맞춤형 추천 설명 (자연어)

ML 작업:

- LLM 텍스트 생성
- Explainability 제공

**⑤ 자연어 감성 검색 (A-5)**

감성 키워드로 영화 검색:

**Input:**

- "우울한데 너무 무겁지 않고 따뜻한 여운이 남는 영화"

**Process:**

```python
1. LLM이 질의를 정서 벡터로 변환
2. 벡터 DB에서 유사 영화 검색 (Top-K)
3. 검색된 영화들을 A-3로 전달하여 만족도 예측

```

**Output:**

- 감성 키워드에 맞는 영화 리스트 + 만족 확률

**ML 작업:**

- RAG 패턴 (Retrieval-Augmented Generation)
- 벡터 유사도 검색

---

**⑥ 그룹 취향 시뮬레이터 (A-6)**

여러 사람의 취향을 종합하여 그룹 추천:

Input:

- 사용자 A, B, C의 취향 벡터

Process:

```python
1. 개별 취향 벡터 비교
   → 공통 선호 요소 추출

2. 합성 취향 벡터 생성

```

Output:

```python
그룹 만족 확률: 65%

"A와 B는 잔잔한 서사를 선호하지만, 
C는 좀 더 역동적인 전개를 좋아합니다.
```

## Test & Iterate - 고객과 어떻게 검증하고 성공을 측정?

### 검증 방법:

1. **정확도 검증**
    - 사용자 5-10명의 실제 리뷰 데이터로 취향 벡터 생성
    - 이미 본 영화에 대한 만족도 예측 vs 실제 평점 비교
    - 목표: 예측 확률과 실제 만족도의 상관관계 0.7 이상
2. **사용성 테스트**
    - 영화 선택 시간 측정 (기존 대비 50% 단축 목표)
    - "추천 이유가 이해되었는가?" 설문 (5점 만점 4점 이상)
    - 그룹 추천 시 실제 만족도 확인
3. **검색 품질 평가**
    - 감성 키워드 검색 결과의 적합성 평가
    - "찾던 분위기의 영화가 나왔는가?" (만족도 80% 이상)

### 성공 지표:

- **만족도 예측 정확도:** 70% 이상
- **영화 선택 시간:** 기존 대비 50% 단축
- **추천 설명 이해도:** 5점 만점 4점 이상
- **재사용 의향:** 80% 이상
